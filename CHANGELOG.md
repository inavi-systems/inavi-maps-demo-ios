# Changelog for the iNavi Maps SDK for iOS

## 0.2.0 - 2019-11-14

### Features
 - `LocationIcon` 기능 추가 (위치 아이콘의 모양과 위치를 변경할 수 있는 기능)

### Bug fixes
 - 특정 지도 레벨에서 셰이프가 사라지는 오류 수정
 - 간헐적으로 축적바의 거리가 비정상 표출하는 오류 수정
 
### Improvements
 - 빠르게 지도 레벨 변경 시 지도 아이콘이 겹치는 현상 개선

## 0.1.0 - 2019-10-29
